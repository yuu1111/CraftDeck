Display Modes:

• Command Button: Executes specified command when pressed

• Player Health: Shows real-time health/max health

• Player Position: Shows X,Y,Z coordinates

• Player Level: Shows experience level and progress


Commands: Use standard Minecraft commands (e.g., /say Hello, /tp @a ~ ~10 ~)

Player Name: Leave empty to use console commands, or specify a player for player-specific commands
